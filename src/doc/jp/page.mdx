# Next.jsのテンプレートプロジェクトへようこそ。

これはすぐに使えるNext.jsのテンプレートプロジェクトです。

開発を簡単にするために、静的生成、サーバーサイドレンダリング、動的ルーティングなど多くの機能が統合されています。


## わく组み
- [Next.js](https://nextjs.org/) は、静的生成、サーバーサイドレンダリング、動的ルーティングなどを含む多くの機能を提供するReactフレームワークです。
- [React](https://reactjs.org/) は、コンポーネント、ステート管理、ライフサイクルなどを含む多くの機能を提供するJavaScriptライブラリです。
- [Tailwindcss](https://tailwindcss.com/) は、ページを迅速に構築するための多くの既製クラスを提供するCSSフレームワークです。
- [radix-ui](https://www.radix-ui.com/) は、ページを迅速に構築するための多くの既製コンポーネントを提供するReactコンポーネントライブラリです。
- [shadcn](https://ui.shadcn.com/) は、ページを迅速に構築するための多くの既製コンポーネントを提供するReactコンポーネントライブラリです。
- [zustand](https://zustand-demo.pmnd.rs/) は、ステート管理、ステート同期、ステート更新などを含む多くの機能を提供するステート管理ライブラリです。
- [next/mdx](https://mdxjs.com/) は、MarkdownファイルをReactコンポーネントに変換できるMarkdownパーサーです。
- [prisma](https://www.prisma.io/) は、データベーステーブルをTypeScriptクラスにマッピングできるORMフレームワークです。
- [t3-env](https://t3-env.vercel.app/) は、環境変数をTypeScriptクラスにマッピングできる環境変数管理ライブラリです。
- [next-auth](https://next-auth.js.org/) は、ログイン、登録、パスワード回復などを含む多くの機能を提供する認証フレームワークです。
- [next-intl](https://next-intl-docs.vercel.app/) は、多言語、多タイムゾーンなどを含む多くの機能を提供する国際化フレームワークです。
- [next-themes](https://github.com/pacocoursey/next-themes) は、ダークモード、テーマ切り替えなどを含む多くの機能を提供するテーマフレームワークです。
- [react-email](https://react.email/) は、ドラッグアンドドロップ、編集、プレビューなどを含む多くの機能を提供するメールエディターです。
- [recharts](https://recharts.org/) は、折れ線チャート、棒チャート、円チャートなどを含む多くの機能を提供するチャートライブラリです。
- [zod](https://zod.dev/) は、型検証、データ変換、データ検証などを含む多くの機能を提供するデータ検証ライブラリです。
