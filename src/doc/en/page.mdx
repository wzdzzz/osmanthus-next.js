# Welcome to the Next.js template project.

This is a ready-to-use Next.js template project.

For easy development, it integrates many features, including static generation, server-side rendering, dynamic routing, etc.


## Frameworks
- [Next.js](https://nextjs.org/) is a React framework that provides many features, including static generation, server-side rendering, dynamic routing, etc.
- [React](https://reactjs.org/)  is a JavaScript library that provides many features, including components, state management, lifecycle, etc.
- [Tailwindcss](https://tailwindcss.com/) is a CSS framework, it provides many features, including utility-first, responsive design, dark mode, etc.
- [radix-ui](https://www.radix-ui.com/) is a React component library, it provides many features, including design system, component library, etc.
- [shadcn](https://ui.shadcn.com/) is a React component library, it provides many features, including design system, component library, etc.
- [zustand](https://zustand-demo.pmnd.rs/) is a state management library, it provides many features, including global state, local state, etc.
- [next/mdx](https://mdxjs.com/) is a markdown parser, it provides many features, including markdown, jsx, etc.
- [prisma](https://www.prisma.io/) is a database toolkit, it provides many features, including database schema, database migration, etc.
- [t3-env](https://t3-env.vercel.app/) is an environment variable management library, it can map environment variables to TypeScript classes.
- [next-auth](https://next-auth.js.org/) is an authentication framework, it provides many features, including login, register, forgot password, etc.
- [next-intl](https://next-intl-docs.vercel.app/) is an internationalization framework, it provides many features, including multi-language, multi-timezone, etc.
- [next-themes](https://github.com/pacocoursey/next-themes) is a theme management library, it provides many features, including light theme, dark theme, etc.
- [react-email](https://react.email/) is an email editor, it provides many features, including drag and drop, rich text, etc.
- [recharts](https://recharts.org/) is a chart library, it provides many features, including line chart, bar chart, pie chart, etc.
- [zod](https://zod.dev/) is a data validation library, it provides many features, including type validation, data conversion, data validation, etc.
