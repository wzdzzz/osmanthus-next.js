# Installation

## 框架
- [Next.js](https://nextjs.org/) 是一个 React 框架，它提供了很多功能，包括静态生成、服务端渲染、动态路由等。
- [React](https://reactjs.org/) 是一个 JavaScript 库，它提供了很多功能，包括组件、状态管理、生命周期等。
- [Tailwindcss](https://tailwindcss.com/) 是一个 CSS 框架，它提供了很多现成的类，可以快速构建页面。
- [radix-ui](https://www.radix-ui.com/) 是一个 React 组件库，它提供了很多现成的组件，可以快速构建页面。
- [shadcn](https://shadcn.com/) 是一个 React 组件库，它提供了很多现成的组件，可以快速构建页面。
- [zustand](https://zustand.surge.sh/) 是一个状态管理库，它提供了很多功能，包括状态管理、状态同步、状态更新等。
- [next/mdx](https://mdxjs.com/) 是一个 Markdown 解析器，它可以将 Markdown 文件转换为 React 组件。
- [prisma](https://www.prisma.io/) 是一个 ORM 框架，它可以将数据库表映射为 TypeScript 类。
- [t3-env](https://t3-env.vercel.app/) 是一个环境变量管理库，它可以将环境变量映射为 TypeScript 类。
- [next-auth](https://next-auth.js.org/) 是一个认证框架，它提供了很多功能，包括登录、注册、忘记密码等。
- [next-intl]() 是一个国际化框架，它提供了很多功能，包括多语言、多时区等。
- [next-themes]() 是一个主题框架，它提供了很多功能，包括暗黑模式、主题切换等。
- [react-email](https://react-email-editor.com/) 是一个邮件编辑器，它提供了很多功能，包括拖拽、编辑、预览等。
- [recharts](https://recharts.org/) 是一个图表库，它提供了很多功能，包括折线图、柱状图、饼图等。
- [zod]() 是一个数据验证库，它提供了很多功能，包括类型验证、数据转换、数据校验等。
